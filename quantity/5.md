# 几何问题

## 平面图形

| 平面图形   | 周长                        | 面积                          |
| ---------- | --------------------------- | ----------------------------- |
| 正方形     | $C=4a$                      | $S=a_2$                       |
| 长方形     | $C=2(a＋b)$                 | $S=ab$                        |
| 圆形       | $C=2\pi r$                  | $S= \pi r^2$                  |
| 扇形       | $C=n^\circ \pi / 180^\circ$ | $S=n^\circ \pi r^2/360^\circ$ |
| 三角形     |                             | $S=ab/2$                      |
| 平行四边形 |                             | $S=ah$                        |
| 梯形       |                             | $S=(a+b)h/2$                  |

## 立体图形

| 立体图形 | 表面积                | 体积           |
| -------- | --------------------- | -------------- |
| 正方体   | $S=6a^2$              | $V=a^3$        |
| 长方体   | $S=2ab+2ac+2bc$       | $V=abc$        |
| 球       | $S=4\pi r^2$          | $V=4\pi r^3/3$ |
| 圆柱     | $S=2\pi rh +2\pi r^2$ | $V=\pi r^2h$   |
| 圆锥     | $S=\pi r^2+\pi rl$    | $V=\pi r^2h/3$ |
