# 几何问题

## 平面图形

| 平面图形   | 周长                        | 面积                          |
| ---------- | --------------------------- | ----------------------------- |
| 正方形     | $C=4a$                      | $S=a_2$                       |
| 长方形     | $C=2(a＋b)$                 | $S=ab$                        |
| 圆形       | $C=2\pi r$                  | $S= \pi r^2$                  |
| 扇形       | $C=n^\circ \pi / 180^\circ$ | $S=n^\circ \pi r^2/360^\circ$ |
| 三角形     |                             | $S=ab/2$                      |
| 平行四边形 |                             | $S=ah$                        |
| 梯形       |                             | $S=(a+b)h/2$                  |

## 立体图形

| 立体图形 | 表面积                | 体积           |
| -------- | --------------------- | -------------- |
| 正方体   | $S=6a^2$              | $V=a^3$        |
| 长方体   | $S=2ab+2ac+2bc$       | $V=abc$        |
| 球       | $S=4\pi r^2$          | $V=4\pi r^3/3$ |
| 圆柱     | $S=2\pi rh +2\pi r^2$ | $V=\pi r^2h$   |
| 圆锥     | $S=\pi r^2+\pi rl$    | $V=\pi r^2h/3$ |

## 三角形性质

### 普通三角形

两边之和大于第三边，两边之差小于第三边

### 直角三角形

#### 勾股定理

两直角边的平方和等于斜边的平方

常见的勾股数有（3，4，5）、（6，8，10）、（5，12，13）、（7、24、25）

#### 斜边中线定理

斜边上的中线等于斜边的一半

#### 相似三角形

如果两个三角形的对应角都相等，那么这两个三角形就是相似的。

对应边（高）的比＝相似比；面积比＝相似比平方；体积比＝相似比立方。

## 最短距离

两点之间直线段最短、点线之间垂线段最短

平面图形常常找对称，立体图形常常先转化成平面图形，再找最短距离。

## 等量最值

### 平面图形

若周长一定，越接近于圆，面积越大（若有四边形约束条件，则为正方形）

若面积一定，越接近于圆，周长越小（若有四边形约束条件，则为正方形）

### 立体图形

若表面积一定，越接近于球，体积越大

若体积一定，越接近于球，表面积越小

## 等比例放缩

若一个几何图形尺度变为原来的 $m$ 倍，则长度变为原来的 $m$ 倍，面积变为原来的 $m^2$ 倍，体积变为原来的 $m^3$ 倍。
