# 论点论据大集合

## 主体

“主体”顾名思义，就是文章中最重要的部分，主体部分在文章中发挥着不可替代的作用。文章的结构是否严谨，条理是否清楚，论证是否严密，论据是否典型，关键在主体部分的写作。文章主体最基本的要求是内容丰富充实、论证深刻具体。

文章主体由分论点、论据、论证构成，因此，写好文章主体就要做到论点鲜明、论据充足、论证合理。下面重点从论点、论据和论证三方面介绍文章主体的写作技巧。

## 论点

### 简介

论点应正确、鲜明，不可模棱两可，具体来说，论点具有以下特点：

①正确性：论点正确是申论文章写作的基本要求，正确的论点是基于给定资料做出的正确分析，也是对客观事物的正确反映。如果论点本身不正确，甚至是荒谬的，文章就难以成立，也很难使人信服。

②鲜明性：赞成什么、反对什么，要明确地表示出来，切忌含糊不清。

③新颖性：在论点正确、鲜明的基础上，文章论点应尽可能新颖、深刻，避免老生常谈、泛泛而谈。

### 分类

在申论文章主体的写作中，论点可以分为原因型论点、影响型论点和对策型论点。

#### 原因型论点

原因型论点即是对问题产生原因的分析。在申论文章写作中，所涉及的重要一点就是要分析问题产生的原因，分析原因需要立足于给定资料，从给定资料中多角度、全方位地挖掘问题产生的原因。

```txt
与时代违和的奇葩校规，何以一直有市场？

奇葩校规虽然看起来与时代违和，但出现在我们面前却一点不令人意外。
首先，奇葩校规是学校间恶性竞争的必然结果。（原因型论点一）……
其次，奇葩校规暗合国人悬梁苦读的传统。（原因型论点二）……
再则，奇葩校规也是家长名校情怀的投名状。（原因型论点三）……与时代违和的奇葩校规，何以一直有市场？
奇葩校规的形成，直接原因当然是校方的成绩冲动，但其背后也隐含着国人强烈的名校情结和不健康的学习观念。（原因型论点四）……
```

#### 影响型论点

影响型论点是对问题可能造成的影响的分析。影响又分为积极影响和消极影响，积极影响包括意义、益处等，消极影响包括危害、弊端等。

```txt
实施利民好政策实现伟大中国梦

习近平总书记说：政策好不好，要看乡亲们是哭还是笑。可见好政策的评判标准，关键在于政策是否利民惠民，能否让公民的生活更美好。好政策对公民的影响是深远的，好政策不仅能够满足公民意愿，更能滋养公民理性，涵养公民德性（影响型总论点）。
好政策能够滋润理性，促进公民的理性思考。（影响型论点一）……
好政策还能够涵养德性，提升公民的自我修养。（影响型论点二）……
好政策不仅满足了公民的意愿诉求，更奏响了中华民族的复兴前奏。（影响型论点三）……
```

#### 对策型论点

对策型论点是为解决问题所采取的相应的措施、做法。对策要合理可行，具备可操作性，对策措施主要从给定资料中提取，也可以结合题目要求，联系实际，从实际生活中获得。

```
寻找乡村文化之根

寻找乡村文化之根，应推进乡村基层文化建设。（对策型分论点一）……
寻找乡村文化之根，应大力发展农村经济。（对策型分论点二）……
寻找乡村文化之根，应均衡发展城乡教育。（对策型分论点三）……
文章主体部分以三个对策型分论点具体指出寻找乡村文化之根应采取的对策措施。
```

#### 并列式论点

就是要求考生在书写论点的时候，以并列的方式呈现，通常表现为单字、单词、单句

```
从“保护文化”到“创新文化”再到逐步实现“输出文化”
【1】注重文化保护，练就文化自信强健之“体”。
【2】促进文化创新，培养文化自信睿智之“脑”。
【3】加速文化输出，塑造文化自信坚韧之“心”。
```

#### 思辨型论点

是近几年考试中常考的考点之一，考生要格外注意，一般表现为两者之间的关系，如

①A 是B 的……

②B 是A 的……

③A 与B 为……

```txt
让文明在城乡之间游走
【1】乡村文明为文明的交流注入新鲜血液
【2】城市文明为文明的交流打造坚硬骨骼
【3】城乡文明互补实现理想生活状态
```

## 论据

### 内涵

论据是指能充分反映事物本质，具有代表性的事例与名言，论据（用什么来证明）是支撑论点的材料，是作者用来证明论点的理由和根据。论据是文章的血肉，也是使文章内容充实丰富的基础。因此，选用的论据应典型且鲜活，还应真实、切合题旨、有说服力。在申论文章写作中，论据可以从给定资料中提取，也可以联系实际，从现实社会和日常积累中选取有针对性的事例或理论，以达到充分说理的目的。

### 分类

#### 事实论据

事实论据是通过对客观事物的真实描述和概括来佐证论点，是证明论点最有说服力的论据，所谓“事实胜于雄辩”就是这个道理。事实论据包括具体事例、概括事实、统计数据、亲身经历等。

```
好政策能够滋润理性，促进公民的理性思考。通过“市长信箱”的建立，公民反映问题的方式更加理性；通过“限塑令”的颁布，公民对待政策的思考更加理性；通过“尾号限行”的实施，公民健康出行的理念更加理性
```

#### 理论论据

理论论据是指来源于实践，并且已被长期实践证明和检验过，断定为正确的观点。理论论据包括名言警句、谚语格言以及自然科学的原理、定律、公式等。

```txt
水的智慧在于其博大宽容。“上善若水，水善利万物而不争”是老子在《道德经》中对水的至高评价（引用老子《道德经》中的原话作为理论论据），最善者的品行，如同水一样，可以滋养与造福万物，却不与万物相争，由此可知水所具有的博大胸襟。“海纳百川，有容
乃大”是对水包容性的概括（引用林则徐题于苏轼的一副自勉联“海纳百川，有容乃大”作为理论论据），
```

## 论证

### 简介

论证是用论据来证明论点的过程，论证的目的在于揭示出论点和论据之间的内在逻辑关系，最终使论点成立。

#### 立论

立论是对一定的事件或问题从正面阐述作者的见解和主张的论证方法

表明自己的态度时，要注意以下三点：

①看法和主张必须是经过认真地思考或者一定的实践，确定其为正确的认识和见解，或者是能切实解决实际问题的主张，从而使读者真正提高对事物的认识。

②必须围绕所论述的问题和中心论点来进行论证。论证必须紧扣论点，首尾一致，在论证过程中，不能任意发挥，或者变换论题。并且每个分论点都要与中心论点有关联，要从属于中心论点，从而增强文章的逻辑性和可读性。

③“立”往往建立在“破”的基础之上。在立论的过程中，需要提到一些错误的见解和主张，加以否定和辩驳，以增强说服力，使阅卷人区分考生的观点和错误观点。

#### 驳论

驳论是以有力的论据反驳别人错误论点的论证方式。驳论有三种方法：反驳论点、反驳论据、反驳论证。一篇驳论文可以几种反驳方式结合起来使用，以加强反驳的力量和说服力。

①反驳论点：直接反驳对方论点本身的片面、虚假或谬误，这是驳论中最常用的方法。

②反驳论据：揭示对方论据的错误，以达到推倒对方论点的目的。

③反驳论证：揭露对方在论证过程中的逻辑错误，如大前提、小前提与结论的矛盾，各分论点之间的矛盾，论点与论据之间的矛盾等。

### 分类

#### 举例论证（例证法）

> [!note]
>
> 作用：有力地论证观点（中心论点或分论点），增强文章的说服力。

举例论证是指列举确凿、充分、具有代表性的事例或数据来证明论点。运用举例论证进行论证时，列举的事实可以有两种形式，即概括总体性事实和枚举个别事实。概括总体性事实的说服力在于事实所体现的普遍性，它是对事实的总体或全局的全面性统计或概括。采用枚举个别事例的论证方式，不要求全面周到，只需枚举几个事例即可。枚举事例要求有一定的典型性，同时也要考虑到经济原则，尽可能做到合并同类例子。

#### 道理论证（引证法）

> [!note]
>
> 作用：可以增强论证的说服力和权威性，激发读者的阅读兴趣。

道理论证即用经典著作中的精辟见解，古今中外名人的名言警句以及人们公认的定理公式等来证明论点。由于道理论证所引用的材料都是被客观实际所证实的科学结论，或是被人们所公认的道理，具有理论的权威性和思想的深刻性，因而具有不可辩驳的力量和说服力。

| 分析方法           | 结构                                             |
| ------------------ | ------------------------------------------------ |
| 影响分析法（积极） | 分论点本身+过渡句+概括现象+分析影响+作出结论     |
| 影响分析法（消极） | 分论点本身+过渡句+概括现象+分析影响+作出结论     |
| 原因分析法         | 分论点本身+过渡句+概括现象+分析**原因**+作出结论 |
| 对策型分析法       | 分论点本身+过渡句+概括问题+对策**分析**+作出结论 |

#### 对比论证

> [!note]
>
> 作用：全面地突出论证观点（中心论点或分论点），让人印象深刻。

对比论证即用正反两方面的论点或论据作对比，在对比中证明论点。对比论证方式的运用范围很广，中与外、古与今、大与小、强与弱等，都适合进行比较，在比较中分析和阐明了两者的差异之后，是非昭然，自然就能够确立论点了。对比可以是两个对象之间的比较，也可以是同一对象自身前后不同阶段之间的比较，前者称为横向比较，后者称为纵向比较。运用纵向对比的论证方式，不能停留在对事物静态判断的层面上，否则，有时会显得说服力不够。运用对比论证要注意几个问题：第一，比较的双方要具备可比性，在选取比较对象上要慎重。第二，要建立合理的参照系。要进行比较，就必须具有合理的共同参照系，没有共同的参照系，两者就无法进行比较。

#### 比喻论证

> [!note]
>
> 作用：生动形象地论证了观点（中心论点或分论点），使文章浅显易懂，易于理解和接受。

比喻论证即用人们熟知的事物作为喻体来证明论点。在比喻论证中，本体和喻体虽然是两类不同的事物，但是它们之间有共同之处，也具有相应的推理关系。运用比喻论证要注意几个问题：一是用来作为喻体的事物，应当是为大家所熟悉的、具体的、浅显的，这样，才能既通俗又生动地说明另一个事物。二是比喻应当贴切、自然，要能恰到好处地说明被论证事物的特点。

#### 假设反推法

论点：宽容

论据：清朝宰相张廷玉面对邻里的造屋之争，主动作诗劝解老母，“千里家书只为墙，再让三尺有何妨，万里长城今犹在，不见当年秦始皇”。张母见信后主动退让三尺，邻居也深感惭愧，退让三尺，最终成六尺巷。

试问，如果张廷玉不是以宽容待人，而是斤斤计较，火上浇油还会有今天的“六尺巷”美谈吗？
