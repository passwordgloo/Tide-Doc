# 结尾六大技巧

## 结尾简介

结尾是文章结构的重要组成部分，是全文内容发展的必然结果，需要对前文提到的观点进行归纳、总结、强调，或提出希望、号召、要求。结尾的好坏直接影响着阅卷者的评分心理。好的结尾应响亮有力，令人警醒，催人奋进。

结尾应在内容上收束全文，突出中心论点；结构上呼应文题，使文章逻辑更加严谨有序；语言上简洁有力，富有启发性和感染力。

## 结尾原则

（一）紧扣主题

结尾与其他各部分有机组合成一篇结构完整的文章，表现同一个主题，所以结尾是为文章主题服务的。结尾只有紧扣主题，才能使文章的论证说理更严谨。

（二）简洁有力

简洁有力是指结尾作为文章的收束部分，不仅要表意完整，而且要言简意赅、干净利索，避免给阅卷人留下拖沓、冗长的印象。

（三）言之有物

言之有物是指结尾也要和主体部分一样，写出具有实际意义的内容。结尾如果空喊口号、无病呻吟，或生硬总结分论点，就会变得乏味、空洞。

## 结尾技巧

### 前后照应

前后照应，指结尾与前文（包括文章的标题、开头等）进行照应。前后照应既揭示了题意，点明了文章中心主旨，又增强了文章感染力，还能使文章结构更加严谨完整。

```txt
以水为师
开头：老子曾说“上善若水”，在“师法自然”的古代，先贤们从水中看到了“善”，悟出了“道”，甚至从中孕育了中华民族最初的哲学雏形（本句讲先贤们从水中获得启发）。在全面建成小康社会的今天，我们提倡以水为师，师水之柔、师水之智、师水之仁，也要师水
之清（本句联系实际，通过“提倡”点出文章的中心论点，照应开头，引出下文）。

结尾：静水流深，在平静的表面下，水蕴含着丰富的哲理（此句是对水的总结，并照应开头的哲理）。现在我国进入全面深化改革的新时期，更应该（“更应该”引导的是具体怎么做，与开头“提倡”形成照应）以水为师，以水的姿态，“居善地，心善渊，与善仁，言善
信，正善治，事善能，动善时”（此处为从水中学到“善”的应用，照应开头的善），加强社会治理和为政修德方面的探索，促进经济的健康发展和社会的和谐稳定，推动全面建成小康
```

### 总结全文

总结全文，指在文章的结尾处对全文加以总结，以强化观点，突出中心。如此结尾，便于阅卷人把握文章主旨，明晰作者的写作意图。

```
水的智慧
主体：1.水的智慧在于其以柔克刚。2.水的智慧在于其宜动宜静。3.水的智慧在于其博大、宽容。

结尾：水是智慧的象征（文题照应，强化论点）。在时空的不断更迭中，水，造就着生命世界的和谐（此处是对水的价值的总结）。它以柔克刚、亦动亦静、博大宽容的特质，气象万千的灵性，带给人类的，是永无止境的点拨，永不枯竭的启迪（此处是对三个分论点的
总结，强化“水的智慧”这一主题）。事实上，水的姿态即人的姿态，水的智慧即人的智慧，人类应该做到与水的和谐相容，在纯粹的状态中，感知水的智慧（此处升华主旨，表达了不仅要了解水的智慧，还要将水的智慧转化成人的智慧，并与水和谐共生的期望）。
```

### 引用作结

引用作结指在结尾处引用与文章中心论点相关的名句、寓言等，以达到深化文章中心主旨的效果。同时，引用富含哲理的名言警句、诗词寓言等作结，既含蓄隽永，又具有一定的文采，还能留下想象的空间，引人思考。

```
青春是用来奋斗的——五四之际致青年
“现在，青春是用来奋斗的；将来，青春是用来回忆的。”（此处引用习近平主席的名句，突出青春的价值，言简意赅，且更有说服力）青春不息，奋斗不止（此句是对前面引用内容的进一步总结，强调突出了“青春是用来奋斗的”这一中心主旨，而且到此戛然而止，言辞
简练、干脆有力、掷地有声）。
```

### 告诫号召

告诫号召，指考生在结尾处重申问题的严重性或发出呼吁号召。告诫号召可以充分引起人们对问题的注意和重视，启迪人们思考，给人以警醒或鼓励，从而达到警示告诫、呼吁号召的目的。此外，以告诫号召结尾还可以增强文章感染力，引起情感共鸣。

```
灾难难免，忧患意识长存
“天有不测风云，人有旦夕祸福”（此处说明福祸无常），人生没有绝对意义上的安全，人类的历史是与灾难相伴并与之抗争的历史（此处说明灾难无常，希望借此警示社会公众提高警惕）。而忧患意识存在的意义在于最大限度降低灾难的危害系数，忧患意识长存，才能确保更持久的安全
```

### 分析型文章结尾（结尾提对策）

让中华文化大放异彩，要创新机制。需要鼓励文化产业和文化项目走出国门，开发海外市场；对待传统文化资源要保护和开发并重，在保护中利用、在普及中弘扬；注重文化创新，不失时机地挖掘时代资源；加强文化人才等。相信在不久的将来，中华文化定会焕发出蓬勃的生命力，为民族复兴、国家富强奠定丰厚根基！

### 对策型文章结尾（结尾重分析）

“理国无难似理兵，兵家法令贵遵行。”做好超限超载执法工作，既有利于维护群众社会安全，又有利于保证经济发展速度，更有助于改善执法环境、建立良好秩序，为建设法治社会、实现依法治国奠定基础。政府带头令行禁止、执法到位，用政府的有形之手带动社会的无形风气转变，一条更加安全通畅的道路会出现在民众面前，通向一个更有秩序的社会和一个更加强盛的中国。
